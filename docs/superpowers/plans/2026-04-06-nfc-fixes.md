## Task 2: NFC-Scan-Button im Registrierungsformular

**Files:**

Ziel: Neben dem UID-Eingabefeld erscheint auf Geräten mit Web NFC API (`'NDEFReader' in window`) ein Scan-Button. Klick startet den Scan, liest die UID des Tags und füllt das Feld automatisch aus. Während des Scannens zeigt der Button einen Ladeindikator.

- [ ] **Schritt 1: Scan-State und Funktion hinzufügen**

In `NfcTagManager.tsx` — füge direkt nach dem `const [savingProd, ...]`-State (oder nach allen anderen States, vor `function addAction`) ein:

```typescript
const [scanning, setScanning] = useState(false)
const [nfcSupported] = useState(() => typeof window !== 'undefined' && 'NDEFReader' in window)

async function scanUid() {
  setScanning(true)
  try {
    const ndef = new (window as any).NDEFReader()
    await ndef.scan()
    ndef.addEventListener('reading', ({ serialNumber }: { serialNumber: string }) => {
      setUid(serialNumber)
      setScanning(false)
    }, { once: true })
  } catch {
    setScanning(false)
  }
}
```

```tsx
// ── Honigernte-Formular ──────────────────────────────────────────
interface HonigernteForm { zargen: number }

function HonigernteFormComp({ value, onChange }: { value: HonigernteForm; onChange: (v: HonigernteForm) => void }) {
  const dec = () => onChange({ zargen: Math.max(1, value.zargen - 1) })
  const inc = () => onChange({ zargen: value.zargen + 1 })

  return (
    <div>
      <p className="text-[12px] font-semibold text-zinc-500 mb-3">Anzahl Zargen</p>
      <div className="flex items-center justify-center gap-6">
        <button type="button" onClick={dec}
          className="w-14 h-14 rounded-2xl bg-zinc-100 hover:bg-zinc-200 text-3xl font-light text-zinc-700 transition-colors flex items-center justify-center">
          −
        </button>
        <div className="text-center min-w-[72px]">
          <p className="text-3xl font-bold text-zinc-900">{value.zargen}</p>
          <p className="text-[12px] text-zinc-400 font-medium">Zargen</p>
        </div>
        <button type="button" onClick={inc}
          className="w-14 h-14 rounded-2xl bg-zinc-100 hover:bg-zinc-200 text-3xl font-light text-zinc-700 transition-colors flex items-center justify-center">
          +
        </button>
      </div>
    </div>
  )
}
```

## Manuelle Verifikation

Nach der Umsetzung auf einem Android-Gerät mit Chrome prüfen:

1. **Registrierungsformular**: "Tag registrieren" öffnen → Scan-Button erscheint → antippen → NFC-Tag halten → UID wird automatisch eingetragen
2. **Scan-Flow Varroa**: Tag scannen → "Varroa Behandlung" wählen → Datum-Karte erscheint → "Behandlung buchen" → Erfolg
3. **Scan-Flow Honigernte**: Tag scannen → "Honigernte" wählen → Zargen-Stepper erscheint → Zahl einstellen → "Honigernte buchen" → Erfolg
4. **Labels**: In der Tag-Liste erscheinen "Durchschau" und "Varroa Behandlung" (nicht mehr "Durchsicht"/"Varroabehandlung")
